# Multiplication Tables Website

This is an interactive website designed to help children learn multiplication tables from 2 to 12 through engaging audio in Canadian English.

## Features

- Interactive multiplication tables from 2 to 12
- Kid-friendly design with colorful elements
- Audio narration in Canadian English for each table
- Play/pause and repeat functionality
- Progress tracking to monitor learning progress
- Responsive design for all screen sizes

## How to Use

1. Select a multiplication table (2-12) by clicking on the corresponding number button
2. View the multiplication table displayed on the screen
3. Click "Play Table Song" to hear the multiplication table in Canadian English
4. Use the pause button to pause the audio if needed
5. Toggle the repeat button to play the audio on loop
6. Your progress is automatically tracked and displayed in the progress section

## Files Structure

- `index.html` - Main HTML file
- `css/styles.css` - CSS styles for the website
- `js/script.js` - JavaScript functionality
- `audio/` - Directory containing audio files for each multiplication table

## Technical Details

- The website uses HTML5, CSS3, and vanilla JavaScript
- Audio files are generated using Google's Text-to-Speech with Canadian English accent
- Progress is saved in the browser's localStorage

## Educational Benefits

This website helps children:
- Memorize multiplication tables through audio repetition
- Visualize multiplication concepts
- Track their learning progress
- Build confidence in mathematics

Enjoy learning multiplication tables with this fun and interactive website!
